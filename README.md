# 📊 Exploratory Data Analysis (EDA) on Sales Dataset

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a retail sales dataset to uncover meaningful insights, patterns, and trends. The goal is to understand business performance and support data-driven decision-making.

---

## 🎯 Objectives

* Understand the structure of the dataset
* Perform data cleaning and preprocessing
* Analyze sales, profit, and regional performance
* Identify key trends and patterns
* Visualize data using charts and graphs

---

## 🛠️ Tools & Technologies Used

* **Python**
* **Pandas** – Data manipulation
* **Matplotlib & Seaborn** – Data visualization
* **Google Colab** – Development environment

---

## 📂 Dataset Information

* Dataset Name: *Superstore Sales Dataset*
* Format: CSV
* Contains information such as:

  * Order Date
  * Region
  * Product Name
  * Sales
  * Profit
  * Category

---

## 🧹 Data Cleaning Steps

* Removed duplicate records
* Handled missing values
* Converted date columns to proper datetime format
* Verified data types

---

## 📊 Exploratory Data Analysis

The following analyses were performed:

### 1. Sales Analysis

* Total sales calculation
* Sales distribution across regions

### 2. Product Analysis

* Top-selling products
* Category-wise performance

### 3. Time-Based Analysis

* Monthly sales trends
* Seasonal patterns

### 4. Profit Analysis

* Profit vs Sales relationship
* Identification of low-profit products

---

## 📈 Data Visualization

Used different types of visualizations:

* Bar charts (Sales by Region)
* Line charts (Monthly Sales Trend)
* Scatter plots (Sales vs Profit)

---

## 🔍 Key Insights

* Certain regions generate higher sales compared to others
* Some products have high sales but low profit margins
* Sales show variation across different months
* Business performance can be improved by focusing on profitable products

---

## ✅ Conclusion

This EDA project helps in understanding the dataset and extracting useful insights that can help businesses improve their strategies and performance.
